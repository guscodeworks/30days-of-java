# 30 Days of Java

Esse repositório foi criado para acompanhar meus estudos em Java durante 30 dias.

A ideia não é fazer algo perfeito logo de começo, mas registrar minha evolução enquanto aprendo a linguagem, pratico lógica de programação e entendo melhor como funciona a estrutura de um programa Java.

## Objetivo

Meu objetivo é estudar Java um pouco por dia, deixar tudo organizado no GitHub e acompanhar minha própria evolução.

Quero usar esse repositório para guardar códigos simples, anotações e pequenos exercícios feitos durante o aprendizado.

## Organização

Cada dia terá uma pasta própria com o código e uma anotação simples sobre o que aprendi.

Exemplo:

```text
day-01/
  HelloWorld.java
  anotacao.md
```

## O que pretendo estudar

Pretendo passar pelos fundamentos do Java, como estrutura básica, variáveis, tipos de dados, operadores, condições, laços de repetição, métodos, arrays, classes, objetos e exercícios de lógica.

## Por que estou fazendo isso?

Estou começando a estudar Java e quero criar o hábito de praticar, anotar e subir meus estudos para o GitHub.

Esse repositório serve para mostrar minha evolução de forma simples, real e organizada.
