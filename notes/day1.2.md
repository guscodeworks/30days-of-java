Explicação TiposPrimitivos

Em Java temos os tipos de variáveis as primitivas:
`byte a = 10;
short b = 15;
int c = 20;
long d = 25L;
//Ponto Flutuantes
float e = 30f;
double f = 35;

//Booleanos
char g = 'B';
boolean h = true || false;
//Escrita
String nome = "Gustavo"; `
