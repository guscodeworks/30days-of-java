## Exercicios

1- Senha correta

Faça um programa em Java que peça uma senha para o usuário.

A senha correta deve ser:

1234

O programa deve continuar pedindo a senha enquanto o usuário digitar uma senha errada.

Quando o usuário acertar, o programa deve mostrar:

Senha correta! Acesso liberado.

R: `Ex1Senha.java`

2- Tabuada

Peça um número para o usuário e mostre a tabuada dele de 1 até 10 usando do-while.

R: `Ex2Tabuada.java`
