## Explicação

Aqui aprendemos o básico de um programa Java.

Aonde teremos o código
`package day1;
public class HelloWorld {
    public static void main (String[] args) {
        System.out.println("Hello World");
    }
}`

Que diz, "Uma classe pública chamada Hello World, com o método Principal(main)
estático(imutável) e não receberá retorno (void) com o parâmetro String(sequência de caracteres) e args(argumentos)"

O package serve para organizar o projeto e evitar bagunça quando tiver muitas classes.

Juntamente com isso temos o método

`System.out.println("Hello World");`

Que vai escrever algo no console com quebra de linha.
